## 🧘 Overview

Serenity Yoga Studio is a modern, responsive landing page designed for yoga studios and wellness centers. The website features a soothing color palette, elegant typography, and smooth animations that embody the peaceful nature of yoga practice.

## ✨ Features

### Hero Section
- Full-screen hero with captivating background imagery
- Animated tagline and compelling headline
- Dual call-to-action buttons for immediate engagement
- Feature highlights showcasing studio benefits

### Weekly Class Schedule
- Interactive tabbed schedule organized by day
- Detailed class information including time, instructor, and level
- Visual indicators for class types (Beginner, Intermediate, Advanced)
- Responsive grid layout for easy browsing

### Instructor Profiles
- Beautiful bio cards with professional photos
- Social media integration
- Specialty and certification highlights
- Hover effects for enhanced interactivity

### Contact Form
- Fully functional contact form with validation
- Real-time input feedback
- Success/error messaging
- Clean, accessible form design

### Additional Features
- Smooth scroll navigation
- Sticky header with scroll effects
- Mobile-responsive hamburger menu
- Scroll-to-top button
- Intersection Observer animations
- Accessibility compliant (WCAG 2.1)

## 🎨 Design Philosophy

The design embraces a **calming and natural aesthetic**:

- **Color Palette**: Soft sage greens, warm beiges, and muted earth tones
- **Typography**: Elegant serif headings (Cormorant Garamond) paired with clean sans-serif body text (Montserrat)
- **Imagery**: Nature-inspired visuals promoting peace and mindfulness
- **Whitespace**: Generous spacing for a breathable, uncluttered feel

## 🛠️ Technologies Used

- **HTML5** - Semantic markup structure
- **CSS3** - Custom properties, Flexbox, Grid, animations
- **JavaScript (ES6+)** - Interactive functionality
- **Google Fonts** - Cormorant Garamond & Montserrat

## 🚀 Getting Started

### Prerequisites
- Any modern web browser (Chrome, Firefox, Safari, Edge)
- A local development server (optional, for best experience)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/serenity-yoga-studio.git
   ```

2. **Navigate to the project directory**
   ```bash
   cd serenity-yoga-studio
   ```

3. **Open in browser**
   - Simply open `index.html` in your preferred browser
   - Or use a local server like Live Server (VS Code extension)

### Using Live Server (Recommended)

```bash
# If you have Node.js installed
npx serve .

# Or with Python
python -m http.server 8000
```

## 🔧 Customization

### Changing Colors
Edit the CSS custom properties in `style.css`:

```css
:root {
  --color-primary: #7C9A92;
  --color-secondary: #E8D5B7;
  --color-accent: #B8860B;
  /* ... more variables */
}
```

### Adding New Classes
Update the schedule section in `index.html` and follow the existing card structure.

### Modifying Instructors
Edit the instructor cards in the `#instructors` section with new images and bio information.
